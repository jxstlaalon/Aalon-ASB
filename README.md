# Aalon Peters | SPEAR Campaign Website

This is the official campaign website for **Aalon Peters**, candidate for **Director of Extracurricular Academic Activities**, running with the **SPEAR** (Students Propelled by Execution, Authenticity & Reform) team.

**Live Site:** [https://voteaalon.ct.ws](https://voteaalon.ct.ws)

## 🚀 Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive Modals:** Custom JavaScript modal system for viewing candidate photos, team logos, and detailed goal breakdowns.
- **Endorsement System:** A dedicated video grid showcasing supporting voices with interactive "is-playing" states and detail tooltips.
- **Accessibility Focused:** Includes ARIA labels, focus-trap logic for modals, and high-contrast styling.
- **Modern UI:** Built with a custom blue and yellow palette (#005ADE / #FECD00) using the *Bebas Neue* and *Outfit* typefaces.

## 🛠️ Technology Stack

- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom properties (variables), Grid, and Flexbox layout systems.
- **Vanilla JavaScript:** Light-weight interaction logic.
- **Assets:** Media served via high-performance CDN (JSDelivr).

## 📁 File Structure

- `index.html`: The core structure and content.
- `style.css`: All layout and component styling.
- `script.js`: Interactive functionality and modal content definitions.
- `robots.txt`: Search engine crawler instructions.
- `sitemap.xml`: Search engine site mapping.

## 💻 Local Development

1. Clone this repository or download the source files.
2. Open `index.html` in any modern web browser.
3. For best results with video transitions, use a local server environment (like VS Code "Live Server").
